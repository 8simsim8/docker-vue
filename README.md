# Docker Vue-cli 3

> Docker-compose for @vue/cli (NODE - VUE3)

Build and run container

```sh
$ docker-compose build
$ docker-compose up -d
```

Create a new vue/cli proyect

```sh
$ docker-compose run --rm node vue create webpack web
```

Copy your proyect in `web/` folder

```sh
$ web/
```

Access project

```sh
http://localhost:8080
```
